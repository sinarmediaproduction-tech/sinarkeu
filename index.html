Sinarkeu adalah aplikasi pencatatan keuangan berbasis web yang berjalan sepenuhnya di browser Anda. Data disimpan secara aman di cloud menggunakan Supabase dan terenkripsi dengan password pribadi Anda.

☁️ Setup Akun & Project Supabase

Supabase adalah platform database PostgreSQL open-source yang menyediakan tier gratis. Sinarkeu menggunakannya sebagai backend penyimpanan data cloud.

Langkah 1 — Buat Akun Supabase
1
Buka supabase.com
Kunjungi https://supabase.com dan klik tombol "Start your project" atau "Sign Up".
2
Daftar dengan GitHub atau email
Disarankan menggunakan akun GitHub untuk kemudahan login. Akun gratis sudah cukup untuk penggunaan pribadi.
Langkah 2 — Buat Project Baru
1
Klik "New Project"
Dari dashboard Supabase, klik tombol New Project.
2
Isi detail project
Name: bebas, misal sinarkeu-kas
Database Password: buat password kuat — simpan baik-baik!
Region: pilih Southeast Asia (Singapore) untuk koneksi lebih cepat
Plan: Free sudah cukup
3
Tunggu project selesai dibuat
Proses pembuatan membutuhkan 1–2 menit. Tunggu hingga status berubah dari "Setting up" menjadi hijau.
Langkah 3 — Ambil URL & Anon Key
1
Buka Project Settings → API
Di sidebar kiri Supabase, klik ikon ⚙️ Settings, lalu pilih API.
2
Salin Project URL dan anon/public key
Project URL — bentuknya: https://xxxxxxxx.supabase.co
Project API Keys → anon / public — dimulai dengan eyJhbGciOiJIUzI1NiIs...

Kedua nilai ini yang dimasukkan ke form setup Sinarkeu.
⚠️ Jangan gunakan service_role key — selalu gunakan anon/public key saja untuk keamanan.
🛢️ Membuat Tabel Database (SQL)

Sinarkeu membutuhkan 4 tabel di Supabase. Jalankan SQL berikut di SQL Editor Supabase.

⚠️ Jalankan semua SQL di bawah ini sebelum menghubungkan aplikasi. Cukup sekali saja.
