📖 Panduan Lengkap Sinarkeu

Selamat datang di Sinarkeu!

Aplikasi ini adalah buku kas digital yang aman, multi‑akun, dan bisa disinkronkan ke cloud. Semua data Anda dienkripsi dengan password pribadi.

🔐 1. Setup Awal
  • Supabase URL & Anon Key – Dapatkan dari dashboard Supabase Anda. Masukkan di form setup pertama kali.
  • Password Enkripsi – Buat password minimal 6 karakter. Simpan baik‑baik, karena tidak bisa dipulihkan. Password ini digunakan untuk membuka kunci aplikasi setiap kali.
👤 2. Manajemen Akun
  • Klik tombol "Ganti Akun" di pojok kanan atas.
  • Setiap akun terhubung ke database Supabase yang berbeda. Cocok untuk memisahkan keuangan pribadi, bisnis, atau keluarga.
  • Untuk menambah akun, buka "Manajer Akun" dari layar kunci atau dari menu ⚙️ Setelan.
  • Setiap akun memiliki password enkripsi sendiri – Anda akan diminta saat beralih akun.
📚 3. Buku Kas
  • Di dalam satu akun, Anda bisa membuat banyak buku kas (misal: "Bulanan", "Investasi", "Proyek A").
  • Gunakan dropdown "BUKU" di header untuk berpindah buku.
  • Kelola buku (tambah/hapus) melalui tombol "Kelola" atau menu ⚙️ Setelan.
  • Setiap buku punya anggaran, transaksi, dan lognya sendiri.
💰 4. Transaksi
  • Tekan tombol "+ Transaksi" untuk mencatat pemasukan atau pengeluaran.
  • Isi jenis, kategori (untuk pengeluaran), deskripsi, dan jumlah.
  • Anda bisa melampirkan foto nota – gambar akan diunggah ke Supabase Storage.
  • Edit atau hapus transaksi dengan menekan tombol ⋮ di kolom Aksi.
  • Gunakan pencarian, filter jenis, dan filter tanggal untuk menemukan transaksi dengan cepat.
📊 5. Anggaran
  • Anggaran Dasar – Template yang berlaku otomatis untuk setiap bulan baru. Atur melalui tombol "Dasar" di panel anggaran.
  • Anggaran Bulanan – Klik "Anggarkan" untuk mengatur anggaran khusus untuk bulan tertentu. Ini akan menimpa Anggaran Dasar hanya untuk bulan itu.
  • Progress anggaran terlihat di panel atas (target, realisasi, sisa).
  • Sistem memberi peringatan visual dan notifikasi Telegram saat anggaran mencapai 80% dan 100%.
📈 6. Laporan & Grafik
  • Panel "Pengeluaran per Kategori" menampilkan diagram donat – klik judulnya untuk membuka/tutup.
  • Tombol "Laporan" membuka laporan bulanan lengkap dengan tabel perbandingan target vs realisasi, serta grafik batang.
  • Laporan bisa diekspor ke PDF dengan tombol di dalam modal.
🤖 7. Analisis AI
  • Fitur ini membutuhkan URL Cloudflare Worker yang Anda deploy sendiri (gratis).
  • Masukkan URL di ⚙️ Setelan → Analisis AI.
  • Setelah dikonfigurasi, klik "Analisis AI" di dashboard, pilih periode dan jenis analisis, lalu dapatkan wawasan dari Groq AI (LLaMA).
📲 8. Notifikasi Telegram
  • Buka ⚙️ Setelan → Notifikasi Telegram.
  • Isi Bot Token (dari @BotFather) dan Chat ID (dari @userinfobot).
  • Setelah disimpan, Anda akan menerima notifikasi untuk setiap transaksi, peringatan anggaran, dan ringkasan harian otomatis pukul 21:00.
💾 9. Backup & Restore
  • Backup Lokal – Simpan snapshot data ke localStorage browser (maksimal 5 versi).
  • Backup Cloud – Simpan ke tabel Supabase, bisa di-restore dari perangkat lain.
  • Google Sheets – Masukkan URL Web App di ⚙️ Setelan untuk mengirim backup ke spreadsheet Google.
  • Semua backup otomatis berjalan setiap hari (untuk cloud) dan dapat dipicu manual.
📦 10. Ekspor & Impor Data
  • Ekspor ke CSV untuk dibuka di Excel, atau JSON untuk arsip.
  • Impor data dari file JSON (duplikat ID akan dilewati).
  • Fitur "Arsipkan JSON & Kosongkan Database" di ⚙️ Setelan – berguna untuk membersihkan data lama dan tetap mempertahankan arsip.
📊 11. Fitur Pendukung Lainnya
  • Kurs USD/IDR – Ditampilkan secara otomatis dari beberapa sumber API.
  • Harga Emas Antam – Bisa diaktifkan dengan API key dari emas.maulanar.my.id (gratis).
  • Zakat Maal – Atur persentase di ⚙️ Setelan, dan kartu zakat akan muncul di dashboard.
  • Log Audit – Semua aktivitas tercatat dan bisa dilihat di bawah tabel transaksi.
📶 12. Mode Offline & Sinkronisasi
  • Aplikasi bekerja offline dalam mode baca‑saja (tambah/edit/hapus dinonaktifkan).
  • Setiap kali koneksi pulih, sinkronisasi terjadi otomatis setiap 30 detik.
  • Tombol "Sinkronisasi" di header memicu sinkronisasi penuh manual (pull semua buku & setelan).

💡 Tips: Gunakan password yang kuat dan simpan di tempat aman. Manfaatkan fitur notifikasi Telegram agar tidak ketinggalan perubahan.
